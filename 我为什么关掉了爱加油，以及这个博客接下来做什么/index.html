我为什么关掉了爱加油，以及这个博客接下来做什么

发表于 2026-06-16 更新于 2026-06-20 分类于 博客
⏳ 约3分钟 · 1237汉字+44词

先说结果：aijia.you 于 2026 年 6 月 1 日停止运营。数据备份了，服务器快照打了，Docker 容器停了，域名现在是一个说明页。

但在解释为什么之前，我想先说一件更真实的事——

付费用户屈指可数

爱加油从上线到关掉，实际的付费用户屈指可数。基本上是几个朋友在用，后台翻一翻，后期连他们的调用量也越来越少了。

同期，我陆续看到一些同行的消息。有人靠 Token 转售月流水做到很大的数字，截图里那种增长曲线，谁都会多看两眼。也有人因为上游渠道走在违法边缘被立案。一个赛道里同时存在这两种极端，说明它还在野蛮期。

我得诚实承认：看到那些月流水，我是眼馋的。不只是眼馋，还有”为什么我做了同样的事，数字差这么远”的实在落差感。

但我也很快意识到几件事。

我接的是 DeepSeek、SiliconFlow、阿里云百炼这些合规上游——这条路注定不会暴利。

真正能把这门生意做起来的人，投入的是 all in 级别的精力——运营、客服、渠道、内容、定价、客户关系，每一块都要持续跑。我之前那种”业余写写公众号、偶尔更新功能”的节奏，和人家根本不是一个量级的玩法。

还有：光靠写公众号是做不成这门生意的。公众号能带来认知，带不来稳定的付费转化。要做大，得做投放、做社群、做 to B 销售。这些事比搭一个网站再写文章推广，要难很多，也更消耗精力。

想清楚这些，反而轻松了。

平台条款是个定时炸弹

除了现实判断，还有一个更硬的原因。

AI API 中转，本质上是转售别人的服务。当上游模型厂商的 ToS 里写着”不得转售”，你的整个商业模型就建在一个法律模糊地带上。我踩过一次边界，让我意识到这不是我想长期承担的风险——尤其是在我还有别的事情要做的时候。

收款同样脆弱。面向国内用户的 SaaS，微信支付和支付宝的接入门槛、Stripe 对国内商家的限制、汇率、提现……每一个环节都是麻烦。我花在处理收款问题上的时间，远比我预期的多。

三件事合在一起，就是一个信号：这个方向的性价比不值得我继续投入。

这个博客本来就不只是爱加油

关掉爱加油之后，我重新看了一遍这个博客。

被读得最多的几篇：万字 AI 简史、BYOK 算力主权、Markdown 格式主权。这些文章指向的方向，从来不是”来买我的 API Key”，而是——把几样底层能力握在自己手里。

这个博客的新世界观

这段时间我反复在想，我真正想写的是什么。最后落在一条判断上：

会过时的，做等等党；不会过时的，现在就学。

具体说：

追某个具体模型（Claude 5、GPT-5、DeepSeek V4）——等等，谨慎追。模型每隔几个月就会有新版本，今天的最优解，半年后可能已经不是答案。

追某家产品（Cursor、Perplexity、各种 AI 工具）——同样，等等。今天的热门工具，明天可能已经被收购（SpaceX 就刚收购了 Cursor）、停更或者被更好的替代。

但英语就不是，现在就该学。五年后还有用，十年后还有用。

这个框架不是反对用 AI 新工具，而是说：追新品测评是旧的流量打法，常青内容才有复利。看一下这个博客被读得最久的几篇，恰好都是常青文，不是新闻。这就是配比该往哪边调的证据。

接下来这个博客的重心：英语 + 打字，以及 AI 时代该自己握住的底层能力。

接下来写什么

「最值得学会的一件事」系列会继续。

之一是 BYOK，之二是 Markdown，之三是做一些我喜欢的小产品。

比如：用打字来练英语。中文母语者的英语输出问题，不是词汇量不够、不是语法不懂，是缺少主动输出的练习。打字是最好的切入点，也是我自己真实在做的事。之后会有一篇讲清楚为什么，以及具体怎么做。

如果你以前是爱加油的用户

如果你用爱加油是为了 BYOK、为了把算力握在自己手里——这条线没断，只是不再需要通过我：

  • 国内模型直连：DeepSeek 开放平台、Kimi API、Qwen API，直接申请，比中转更稳
  • 海外模型：OpenRouter 是目前最正规的聚合方案
  • BYOK 实战教程：如何使用自己的 API Key 调用 Cline 已更新为中立版本，可以直接参考
