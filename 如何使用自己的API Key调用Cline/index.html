如何使用自己的API Key调用Cline（BYOK实战指南）

发表于 2026-03-27 更新于 2026-06-17
⏳ 约2分钟 · 563汉字+78词

【编者注】 本文原版以「爱加油（aijia.you）」作为 API 示例平台，该平台已于 2026 年 6 月停止运营。本文已更新为产品中立方案，所有 API 配置示例改用官方/主流平台。

本文介绍如何用自己的 API Key 驱动 Cline，把算力主权握在自己手里。

延伸阅读：BYOK 算力主权：AI 时代最值得学会的一件事（之一）

为什么 Cline + BYOK 值得学

Cline 是 VS Code 里最好用的 AI 编程助手之一，但默认订阅价格不低。BYOK 让你直接用官方 API，按量付费，成本往往低 5–10 倍，且不受平台账号封禁影响。

核心优势：

1. 稳定，不封号

直接对接国内官方 API（DeepSeek、Kimi、通义等），无中间层，网络稳定，账号风险极低。

2. 语言无障碍

Cline 支持设置中文为偏好语言，和 AI 的沟通可以全程中文。

3. 模型自由

可以灵活搭配国内外顶级模型：

  • 国产模型：DeepSeek、Qwen、Kimi——性价比高，国内直连
  • 海外模型：Gemini、GPT-4o 等——通过 OpenRouter 聚合接入

核心配置：三步让 Cline 为你工作

第一步：安装 Cline 插件

在 VS Code 扩展市场搜索 Cline，安装后侧栏出现机器人图标。

第二步：获取 API Key

选择一个平台注册并创建 Key：

DeepSeek 开放平台 https://api.deepseek.com/v1 日常首选，价格极低，中文好
月之暗面 Kimi https://api.moonshot.cn/v1 长上下文，读大文件场景
通义千问 https://dashscope.aliyuncs.com/compatible-mode/v1 阿里生态，模型丰富
OpenRouter https://openrouter.ai/api/v1 海外模型聚合（Gemini/GPT/Claude）

  • OpenRouter：在 openrouter.ai 搜索模型名，直接复制 slug

进阶：Plan Mode 和 Act Mode 用不同模型

Cline 支持为计划阶段和执行阶段分别配置模型，是省钱的关键：

  • Plan Mode（讨论需求、拆分任务）：用推理型模型，如 deepseek-reasoner 或 Gemini 2.5 Pro
  • Act Mode（具体写代码、改文件）：用快速型模型，如 deepseek-chat 或 Gemini Flash

这个组合在质量和成本之间取得很好的平衡。
